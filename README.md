# Voxnet_lidar
An on going TF implementation on Voxnet to deal with LiDAR pointcloud.

## Reference Paper
```txt
VoxNet: A 3D Convolutional Neural Network
```

## Dataset
[SydneyUrbanObjectDataset](http://www.acfr.usyd.edu.au/papers/SydneyUrbanObjectsDataset.shtml)
